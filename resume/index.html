TodoPath

2016

Github Architektur Dokumentation

Einfache native Android Todo-Applikation entstanden im Rahmen des Studiums zum Erstellen und Verwalten von Todos. Flixget

2016
Javascript, NodeJS + Express, Redis, HTML, SCSS, JQuery
Github Design-Demo Dokumentation

Flixget ist eine Webanwendung, welche beim Aufruf einen zufälligen auf Netflix verfügbaren Film vorschlägt. Hierbei kann nach Netflix-Region gefiltert werden. Benutzer können sich ebenso registrieren und Filme als gesehen markieren, um so keine wiederholten Empfehlungen zu bekommen.

Flixget ist nach dem SOA-Prinzip strukturiert. Der Dienstgeber bietet eine stateless RESTful API an welche der Dienstnutzer konsumiert. Ursprünglich enstanden im Rahmen des Studiums. Der Dienstgeber (NodeJS, Redis) ist komplett fertig, der Dienstnutzer leider nur im Ansatz (Benutzerauthentifizierung fehlt). Redis war als primäre Datenbank vorgegeben. Ich arbeite gerade an einer Re-Implementierung mit AdonisJS (NodeJS Framework) + VueJS Frontend. Die verlinkte Design-Demo zeigt das aktuelle Design mit Mobile-First Ansatz.
